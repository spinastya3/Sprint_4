# Автотесты для сервиса "Самокат" (Версия 2.0)

В этом проекте я автоматизировала проверку сайта "Самокат". Проект был переработан: я перевела тесты со старого Selenium на **Selenide** и **JUnit 5**, что сделало их быстрее и стабильнее.

## 🛠 Технологии и версии:


| Технология | Версия |
| :--- | :--- |
| **Java** | 11 |
| **Selenide** | 6.19.1 |
| **JUnit** | 5.10.2 (Jupiter) |
| **Allure** | 2.24.0 |

## 🎯 Что проверяют эти тесты:
* **Заказ самоката**: Проверка того, что форма заказа работает с разными наборами данных (использование `@ParameterizedTest`).
* **Раздел FAQ**: Проверка всех 8 вопросов и ответов в выпадающем списке на главной странице.
* **Поиск заказа**: Тест создает заказ, запоминает его номер и пробует найти его через поиск на главной (**End-to-End сценарий**).
* **Проверка ошибок**: Валидация полей ввода (например, некорректный телефон).
* **Логотипы**: Проверка редиректов. Логотип "Яндекс" открывает новую вкладку, а "Самокат" возвращает на главную.

## ⚙️ Как всё устроено внутри:
* **Page Object Model + Fluent Interface**: Тесты написаны в стиле цепочек методов, что делает их читабельными.
* **BaseTest**: Вся логика настройки браузера, очистки кук и закрытия лишних вкладок вынесена в базовый класс.
* **Allure Reports**: Каждый шаг теста залогирован аннотацией `@Step`.

### ⚠️ Найденные баги (тесты могут падать):
1. **Тест на дату (1600 год)**: Сайт ошибочно разрешает аренду в далеком прошлом.
2. **Тест на логотип Яндекса**: Ссылка ведет на **dzen.ru**, что может не совпадать с ожидаемым `ya.ru` в зависимости от версии стенда.
